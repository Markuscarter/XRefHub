Get Your Extension ID

Step 1: Open Chrome Extensions Page

Copy and paste this URL in your browser:

chrome://extensions/

Step 2: Enable Developer Mode

Toggle the "Developer mode" switch in the top right corner of the page.

Step 3: Find Your Extension

Look for "Xrefhub" in the list of extensions.

Step 4: Copy the Extension ID

Under the Xrefhub extension, you'll see an "ID" field. Copy that ID (it looks like a long string of letters and numbers).

Example: The ID will look something like: abcdefghijklmnopqrstuvwxyz123456

Step 5: Create OAuth Client ID

Now go to Google Cloud Console and:

  1. Select your project (or create a new one)
  2. Go to "APIs & Services" → "Credentials"
  3. Click "Create Credentials" → "OAuth 2.0 Client IDs"
  4. Choose "Chrome Extension" as application type
  5. Enter your extension ID from step 4
  6. Click "Create"
  7. Copy the new client ID

Step 6: Update Manifest.json

Replace the client ID in your manifest.json file with the new one you just created.
